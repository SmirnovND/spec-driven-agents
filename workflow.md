# Жизненный цикл работы агента

Этот документ описывает стандартный сценарий работы AI-агента,
использующего спецификации как источник истины.

---

## 1. Получение задачи

- Пользователь указывает входную спецификацию
- Формулирует бизнес-изменение

---

## 2. Анализ

- Агент загружает entry-point спецификацию
- По ссылкам строит дерево зависимостей
- Определяет scope изменений

---

## 3. Планирование

- Создаётся change-plan файл
- Фиксируются изменения:
    - в спецификациях
    - в коде
- Используются чек-листы

---

## 4. Реализация

- Изменения применяются по шагам
- Сначала — спецификации
- Затем — код

---

## 5. Завершение

- Все пункты плана отмечены
- Спецификации и код синхронизированы
- Агент готов к следующей задаче
